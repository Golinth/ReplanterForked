# Replanter Forked

Replanter Forked is client-side only mod for harvesting and replanting crops by holding right click.

## Building
Make sure you have OpenJDK JRE 26 installed.  
Update [gradle.properties](/gradle.properties) to a desired Minecraft version if necessary.
```
./gradlew publishToMavenLocal
```
The output library will be written to `build/libs`.
